### ✨ Overview

DeepslateMC is a Minecraft Server software built as a fork of Leaf

* **🧩 Added features**
  New features directly in the server software to replace everyday plugins.
* **⚙️ Fully customizable**
  You can configure all features to your desire including the messages.
* **🔓 Free and open source**
  Try DeepslateMC for free and check out the source code on GitHub.
